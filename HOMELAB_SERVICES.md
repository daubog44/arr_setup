# 🚀 HaaC Homelab Services

Benvenuto nel tuo Homelab GitOps Enterprise-Grade (HaaC).
Questo cluster Kubernetes (K3s) è completamente automatizzato e gestito da ArgoCD. Include un ecosistema di app per l'observability, la sicurezza, la gestione del sistema e il media streaming.

Ecco la mappa completa di tutti i servizi installati e raggiungibili:

---

## 🛡️ Gestione e Sicurezza

1. **ArgoCD** - Il cuore GitOps. Gestisce e sincronizza ogni manifesto dal tuo repository Git al cluster in tempo reale. (`/k8s/platform/argocd`)
2. **Authelia** - L'Identity Provider e sistema Single Sign-On (SSO). Protegge l'Homelab dietro 2FA (OIDC supportato). (`app.authelia`)
3. **Traefik** - L'Ingress Controller primario preinstallato in K3s. Inoltra il traffico in base all'host, agganciandosi ad Authelia e Cloudflare.
4. **Cloudflared** - Crea un tunnel Zero Trust out-bound sicuro per non esporre alcuna porta del router, integrato nativamente via ingress. (`app.cloudflared`)
5. **SealedSecrets** - Controllore crittografico asimmetrico. Protegge i segreti su Git decifrandoli soltanto localmente nel cluster. (`app.sealed-secrets`)
6. **Gluetun** - Container VPN proxy per scaricare file in anonimato assoluto (attivo per il downloader). (`media.downloaders`)

## 🖥️ Portali e Dashboard

7. **Homepage** - La tua landing page personale con tutti i segnalibri dei servizi e lo stato real-time di container e macchine virtuali. (`haac-stack.homepage`)
8. **Headlamp** - Una freschissima UI Web per amministrare il cluster Kubernetes senza usare il terminale. Accesso SSO integrato! (`app.headlamp`)
9. **Semaphore (Ansible AWX)** - Piattaforma UI per eseguire playbook Ansible. Usata per Restore database automatizzati e gestione nodi hardware. (`mgmt.semaphore`)

## 📊 Observability (Logs, Metriche, Allarmi)

10. **Kube-Prometheus-Stack** - Il monitoring definitivo. Gestisce l'intero ecosistema di metriche. (`app.kube-prometheus-stack`)
11. **Grafana** - Visualizzatore di dashboard pazzesche con Single Sign-On (OIDC). Precaricate cruscotti per Trivy, Metriche e Log.
12. **Prometheus / AlertManager** - Raccoglie metriche vitali dal cluster e dalle app ed innesca gli alert.
13. **Grafana Alloy** - Scraper agnostico moderno per le pipeline di log ed event tracing verso Loki. (`app.alloy`)
14. **Grafana Loki** - Il motore di persistenza nativo per i log del cluster, visualizzabile direttamente in Grafana. (`app.loki`)
15. **Node Problem Detector** - Sonda i kernel di Proxmox per scovare problemi hardware/software a bassa latenza (OOM Kills, Kernel Panics).
16. **Ntfy** - Sistema geniale di push notifications auto-hostato. Usato da ArgoCD e l'Alertmanager per segnalare anomalie sul tuo telefono. (`mgmt.ntfy`)

## 🔐 Security Scanning & Testing

17. **Trivy Operator** - Scansiona instancabilmente Kubernetes alla ricerca di vulnerabilità, falle RBAC e configurazioni errate.
18. **Falco** - IDS di Kernel level (eBPF). Monitora se qualcuno avvia una shell o tocca file sensibili all'interno dei pod. (`app.falco`)
19. **Kured** - Automation agent che monitora i nodi worker ricaricandoli dolcemente (cordon e drain) se ci sono update critici in attesa del riavvio.
20. **Litmus** - Piattaforma di Chaos Engineering cloud native, per spegnere pezzi di infrastruttura a caso e verificare come reagisce. (`app.litmus`)
21. **RenovateBot** - Robot automatizzato esterno che controlla se le Chart e le immagini Docker hanno nuove release, generando Push Request nel repo. (`/renovate.json`)

## 💾 Storage & Database

22. **Longhorn** - Il volume storage replicato per K3s. Assicura che i dati (DB di Authelia, SQLite, ecc.) sopravvivano alla morte di un server. (`app.longhorn`)
23. **PostgreSQL / SQLite** - Database usati primariamente da componenti core come Authelia, Semaphore e il core system K3s stesso.
